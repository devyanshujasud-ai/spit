## 🌎 Overview

**Sentinel AI** is a state-of-the-art environmental intelligence dashboard engineered to track and predict global catastrophes, analyze ecosystem degradation, and provide decision support tools for incident commanders. The platform merges satellite earthdata simulations with interactive 3D visualizations, empowering operators to respond proactively rather than reactively.

## ✨ Key Features

- **🌐 3D Topographic Calamity Map:** Fully interactive, mathematically generated perlin-noise terrain globe mapped with multi-variable geological and climatic stress points.
- **🔴 Priority-Ranked Anomaly Feed:** Real-time data pipeline running Z-score isolation algorithms to rank severity of impending crises.
- **📈 Advanced Temporal Modeling:** Dynamic forecasting visualizing future climatic trajectories within bounds of confidence testing.
- **💬 Context-Aware Query Engine:** Direct communication interface with Sentinel AI Core for rapidly extracting actionable briefings and recommended mitigation procedures.
- **🚀 High-Performance React Engine:** Fully synthesized within a custom glassmorphism styling layer running efficiently on Vite.

## 💻 Tech Stack

- **Framework:** React 18, Vite
- **Web 3D & Data-Viz:** Three.js, Recharts
- **Iconography:** Lucide-React
- **Styling:** Custom Glassmorphic CSS Engine

## 🚀 Getting Started

To launch Sentinel AI locally, follow these simple setup instructions.

### 1. Installation

Ensure you have Node installed, then run the installer:
```bash
npm install
```

### 2. Execution

To fire up the intelligence pipeline and spin up the frontend on your local development machine:
```bash
npm run dev
```

The system will report systems nominal and bind to `http://localhost:5173/`. Connect and input operator credentials to view the command dashboard.
